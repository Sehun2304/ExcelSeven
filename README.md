# ExcelSeven
내일배움캠프 7.17.-24. 엑셀세븐 팀과제 

#### 엑셀세븐 팀원
고은아, 박세훈, 박진성, 박정은, 김장원

#### 구현한 기능
1. 회원가입, 로그인
2. 로그인 사용자 프로필 업데이티, 비밀번호 변경
3. 게시글 작성, 조회,수정, 삭제
4. 댓글 작성, 수정, 삭제
5. 게시글, 댓글 좋아요
